# coconut-prelude

An implementation of [Haskell's Prelude](https://hackage.haskell.org/package/base-4.10.1.0/docs/Prelude.html) in Python using [Coconut](http://coconut-lang.org/).

## Installation

```
pip install coconut-prelude
```

## Usage

```python
from prelude import *
```
